Welcome to GoonAuth

GoonAuth is a simple and easy way to new users as members of the SomethingAwful forums using an easy to implement OAuth2 based API.

Register Today

For Users

Once registered and verified, an (editable by you) profile will be available for GoonAuth enabled websites to use to be able to quickly retrieve information about your various other accounts (such as your Steam profile or Xbox Live gamertag).

For Developers

GoonAuth implements a standards compliant OAuth2 provider which allows you to authenticate and verify your users as SomethingAwful members quickly and easily. Simply register an application in your settings page and read up on our API documentation and example apps provided to you in our GitHub repository.

Open Source

GoonAuth is completely opensource and available on our GitHub repository. Feel free to report any bugs or feature requests there. We are also accepting pull requests and other contributions as well.
